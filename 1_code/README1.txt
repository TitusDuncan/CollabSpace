CollabSpace Midterm MVP (Flask + SQLite)

1) Create and activate a virtual environment
   python3 -m venv .venv
   source .venv/bin/activate

2) Install dependencies
   pip install -r requirements.txt

3) Run the app
   python app.py

4) Open in browser
   http://127.0.0.1:5000/

Uploads are stored in 1_code/uploads/.
Database is stored as 1_code/collabspace.db (SQLite).
