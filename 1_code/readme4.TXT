CollabSpace — Data Collection (Module 8)

Status:
No external data collection is required for this project at this stage.

Reason:
CollabSpace is an information system where the primary data is created directly by users:
- Projects
- Assets (uploaded media)
- Versions
- Presentation pages

Current approach:
- Data is entered through the UI forms and stored in SQLite
- Demo content is created by uploading local sample image/audio files

Optional future enhancement (not required for this course project):
- Importing metadata in bulk (e.g., reading tags/notes from a CSV)
- Migrating from local storage to cloud storage for scalability

Notes:
If any collection/import scripts are added later, they will be included in this folder along with run steps.
